# 🔥 Meta Ads Performance Analyzer

A machine learning-powered web application that analyzes digital ad performance and identifies potential issues using predictive analytics and data-driven insights.

---

## 🚀 Features

* 📊 Analyze ad performance using key metrics (CTR, CPC, Conversion Rate)
* 🤖 Predict ad issues using a trained Random Forest model
* 💡 Generate actionable insights for improving ad performance
* 📂 Upload CSV files for bulk analysis
* 📈 Interactive dashboard with visualizations
* 🎯 Detect issues like:

  * Creative issues
  * Audience targeting issues
  * Conversion issues
  * Well-performing ads

---

## 🧠 How It Works

1. Input ad data (impressions, clicks, spend, conversions)
2. Model processes the data using a trained classifier
3. System predicts performance category
4. Metrics are calculated (CTR, CPC, CVR)
5. Insights are generated for decision-making
6. Dashboard visualizes performance patterns
